UI ELECTRIC VFX

This package includes stylized electric VFX shaders for UI and TextMeshPro. It supports URP and Built-in pipelines and is fully compatible with TextMeshPro (tint, italics, strikethrough, underline).

Shader List and Parameters:

1. UIElectricity.shader
- ColorIntensity: Intensity of the tint color
- Frequency: Number of waves per unit
- Width: Width of the electric ray
- Border fade: Falloff near borders
- Amplitude: Displacement amount
- Tileable: Makes the effect repeat seamlessly
- Variation seed: Controls noise pattern variation

2. ElectricImageUI.shader
- ColorIntensity: Intensity of the tint color
- MipLevel: Controls aura appearance (requires mipmaps)
- Scale: Size of the effect
- Displacement: Distortion power
- InnerFill: Intensity of core fill
- RayThreshold: Cutoff for ray visibility

3. ElectricText.shader
- ColorIntensity: Intensity of the tint color
- Scale: Size of effect
- RayThreshold: Outer ray cutoff
- InnetRayThreshold: Inner ray cutoff
- InnerFill: Intensity of the core fill
